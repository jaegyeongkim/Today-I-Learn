개요: SSAFY만의 취업 커뮤니티 어플

기술스택:

- Front: 안드로이드 스튜디오, 코틀린
- Back: Node.js, MySQL

기능

```bash
- 로그인, 회원가임
  기수, 지역, 이름, 현재 거주 지역, 현재 상태(공개 여부), 희망 기업 / 현재 직장, 사진 올리기
- 채팅
  자유롭게 떠드는 방(자소설 닷컴), DM
- 게시판
  익명 게시판, 스터디 모집, 자유 게시판, 댓글
- 프로필
  기수, 지역, 이름, 현재 거주 지역, 현재 상태(공개 여부), 희망 기업 / 현재 직장
- 알림
  개인 DM, 댓글
```
