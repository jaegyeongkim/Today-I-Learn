## 2020-11-26

Git 강의를 들었는데 신기한 기능들이 많았던 거 같다.

Git에서 내가 사용하는 기능은 status, add, commit, merge, push, branch 정도였는데 의미나 기능을 모르고 주먹 구구식으로 사용하지 않고 기능의 의미를 알고 할 수 있어서 좋다.

오늘 배운 거는 Log 기능과 cmd 창에 띄우는 거를 Custom 해서 보기 좋게 만드는 거 그리고 Pandas나 SQL 처럼 직관적인 명령어로 commit이나 내용을 필터링할 수 있다는 거

이 강의를 듣는 게 땅으로 떨어지지 않고 내 머리로 잘 들어오면 좋겠다용
