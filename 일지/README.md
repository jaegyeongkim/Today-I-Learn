모의면접은 떨어진 곳으로 보여니까 의욕이 없었고 3명이서 압박면접으로 하니까 의욕이 없었던 거 같음. 조급하지 않게 하는 거를 목표로 했는데 저번보다는 낫지만 여전히 튀어나가듯이 대답을 하고 있다. 여유를 가지고 면접에 임해야겠다.

그리고 설명을 말로 하는 연습을 미리 해둬야겠다. 말로 하려니까 계속 멈추는 거 같음.

오늘도 너무 고생 많았다. 오늘 할 수 있었던 최선을 다한 거 같다.

그리고 얼른 노트북을 사야할 거 같네. 노트북 이관도 시키고 그러려면 미리 세팅해놔야하니까 빨리 사두자. 일기로 적지 말고 CS로 채워넣고 싶다.
